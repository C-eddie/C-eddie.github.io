Windows如何安装和使用Python？

Python虽然是一门编程语言，但下载使用其实和普通软件没什么区别。

[TOC]

简单介绍Python

Python是一门相较于其他编程语言来说，相对简单的编程语言。维基百科对Python的描述是：解释型、高级编程、通用型编程语言，由吉多·范罗苏姆创造，第一版发布于1991年。可以视之为一种改良（加入一些其他编程语言的优点，如面向对象）的LISP。Python的设计哲学强调代码的可读性和简洁的语法（尤其是使用空格缩进划分代码块，而非使用大括号或者关键词）。相比于C++或Java，Python让开发者能够用更少的代码表达想法。不管是小型还是大型程序，该语言都试图让程序的结构清晰明了。

对于上面一整段话，只需要记住第一句就可以了。

如何安装Python

Python在不同系统下安装，有不同的操作步骤，今天我们只针对Windows进行分享。Python安装和普通软件安装基本没什么区别，可能会多1～2个步骤。主要分为三个步骤：

  1. 到官网下载并安装Python到电脑上 （按默认路径，并选择Add Python to PATH)
  2. 让电脑知道你把Python下载到哪了 (有的电脑笨，需要手动进行一些操作，下面会解释)
  3. 安装一个用来写Python代码的文本编辑器

1：到官网下载并安装Python到电脑上

这一步是三个步骤里最简单的一步。官网的网址我已经链接到官网这两个字上了，点击即可。由于官网是全英文，如果进到官网里不知道哪里选择下载，这篇文章不会在这里帮助你，编程语言绝大部分是由英文构成，如果连下载都不知道在哪里，后边的编程基本是没路可走的。但也别灰心，这里也有方法：下死功夫，长期浸泡在英文技术网站里。

下载好安装包后，就要安装到电脑上，这里要注意的是：按照默认路径，不要擅作主张更改安装的路径。一定要勾选Add Python to PATH，不然后期会出很多麻烦。这两点在上边有说过，这里在啰嗦 一下。

好，到这里第一步就圆满完成了。

2：让电脑知道你把Python下载到哪了

这一点很重要。有的电脑很聪明，当你按照上面的步骤操作安装了之后，电脑会知道你把Python安装到哪里了。有的电脑脑袋就不那么灵光了，需要你手动进行一些操作（很简单）来告诉电脑Python的具体位置。如果电脑不知道Python的位置，而你也没有告诉它，那么你就没法打开Python，也就没法用Python来实现你脑中的想法了。

先分享第一种情况：电脑知道Python的具体位置。那么怎么打开Python呢？这时候要用到一种叫命令提示符的东西，这是什么呢？在这里先不进行分享，只要按照步骤进行操作就好。

  • win键+R，输入cmd，敲回车。打开的这个东西就叫命令提示符
  • 输入Python，如出现>>>的符号，即说明你的电脑很聪明，知道你把Python安装到哪里了

到这里，如果一切很顺利，那么说明，你的电脑很聪明。如果没有出现>>>，那么你就需要手动进行一些操作，来告诉你的电脑。不用担心，非常简单。我们下面分享第二种情况。

  • 你来找Python的安装位置
  • 告诉电脑

怎么找？Python默认安装路径在C盘，如果你能记得住安装路径，那就可以直接告诉电脑了；记不住的话，在C盘里搜索Python，找到并打开以Python打头的文件夹，再找到文件Python。

找到Python的位置后，怎么告诉电脑呢？还记得之前的命令提示符吗？它是我们跟电脑交流的中间人哦，没有命令提示符，我们是没法跟电脑交流的。所以还要用到它，打开命令提示符，将Python的完整路径输入进去，比如我的Python位置是C:\Python35\python，输入进去后，敲击回车，出现>>>没有？

如果可行，就说明你已经知道如何访问Python了。

3：安装一个用来写Python代码的文本编辑器

到这里，可能有的同学会有疑问，刚才你不是说过怎么成功访问Python了吗？既然在没有文本编辑器的情况下可以访问，那么为什么要下载文本编辑器？

刚才在命令提示符里，虽然成功访问了Python，也能写代码并可以成功执行，但这种访问Python的方式有一个致命的缺陷：没法保存你已经写好的Python代码。不知道你有没有见过程序员写的代码，个人最开始感觉花10年也不可能完成啊！足可以想象代码的数量有多么庞大。一个项目可能花几个星期，几个月也有可能。如果不能保存，那程序员这个职业做的好的，应该是电脑最好的，开机时间维持最久的那个人。

这时候我们就需要用到文本编辑器了，就像游戏里“保存进度”的作用一样，有了它，我们便可以随时随地继续之前保存的编程工作了。当然，文本编辑器还有其他作用，如语法高亮，查询代码等等，这里不展开，也没必要展开，后续大家用文本编辑器开始写程序后会自然而然的了解。

如何使用Python

上述所有的步骤你都完成了之后，这一步其实已经完成了。我们已经可以开始用Python来写东西了，剩下的工作就是了解Python使用过程中的规则，怎么更好的使用Python。
